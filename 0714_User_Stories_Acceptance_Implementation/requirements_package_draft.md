# 需求文件包初稿（Requirements Package Draft）

## 1. 專案與範圍摘要
本專案為智慧課程分組與跨系組隊媒合系統，目標是在不連接正式資料庫與正式登入的前提下，讓使用者先體驗學生填寫技能、查看隊伍、申請加入與組長建立任務的核心流程。

## 2. 角色與利害關係人摘要
- 一般學生：填寫技能與偏好、查看隊伍、提交加入申請
- 專案組長：建立任務、查看任務狀態、更新流程
- 授課老師：查看簡易分組規則與隊伍摘要
